# eCommerce Project

This is an eCommerce project built using [React](https://reactjs.org/) and [Redux](https://redux.js.org/). It showcases a simple online store where users can browse and purchase items.

## Features

-   Browse and search for items
-   Add items to the shopping cart
-   Update quantities and remove items from the cart
-   Checkout and complete the purchase
-   View order history

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/kalczugag/ecommerce.git

    ```

2. Navigate to the project directory:

    ```bash
    cd ecommerce
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Start the development server and json-server:

    ```bash
    npm run start:server
    ```

    ```bash
    npm start
    ```

5. Open the application in your browser:
   http://localhost:3000
